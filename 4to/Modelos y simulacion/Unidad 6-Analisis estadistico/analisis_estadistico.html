Análisis estadístico de datos simulados

En general a la hora de realizar simulaciones, tenemos datos obtenidos del sistema real que luego de ser analizamos decidimos representar estos datos de acuerdo con cierta distribucion de probabilidad. Pero estas dependen de ciertos parametros que la determinan (λ\lambdaλ para la exponencial)y ademas existen ciertas medidas (Esperanza, Varianza, max, min, etc.). Y como por lo general se desconoce la distribucion exacta de X, se estiman estos parametros a partir de la muestra.

Un estimador es una funcion que depende de una muestra de valores. Y tiene la siguientes caracteristicas:

  • Insesgabilidad: Es insesgado si E[θ^]=θE[\hat{\theta}]= \thetaE[θ^]=θ
  • Consistencia: Si al aumentar la muestra el estiamdor se aporxima al valor real
  • Eficiencia: menor varianza, mayor eficiencia.
  • Suficiencia: utiliza toda la información obtenida de la muestra.

Error cuadrático medio de un estimador

El error cuadrático medio de un estimador θ^\hat{\theta}θ^ esta definido como:

ECM(θ^,θ)=E[(θ^−θ)2]=Var(θ^)+(E(θ^)−θ)2ECM(\hat{\theta}, \theta) = E[(\hat{\theta} - \theta)^2] \\ = Var(\hat{\theta}) + (E(\hat{\theta}) - \theta)^2 ECM(θ^,θ)=E[(θ^−θ)2]=Var(θ^)+(E(θ^)−θ)2

Si el estimador es insesgado su ECM es igual a la varianza.

Estimadores en simulaciones

Muchas veces queremos saber cuantas muestras de la poblacion necesitamos para poder obtener un estimador de la media con un ECM menor que cierto ddd. Que deberia satisfacer σ2n<d\frac{\sigma^2}{n} < dnσ2​<d. Pero no siempre sabemos el valor de σ\sigmaσ, por lo que lo reemplazamos con el estimador S2(n)S^2(n)S2(n). Lo que implica qe ambos estimadores deberan ser calculados en cada iteracion con cada nuevo elemento de la muestra que se genera. Por lo que es util tener definiciones recursivas de cada estimador, para que el calculo sea mas eficiente.

X‾(n+1)=X‾(n)+Xn+1−X‾(n)n+1S2(n+1)=(1−1n)S2(n)+(n+1)(X‾(n+1)−X‾(n))2\overline{X}(n +1) = \overline{X}(n) + \frac{X_{n+1} - \overline{X}(n)}{n+1} \\ S^2(n+1) = (1 - \frac{1}{n})S^2(n) + (n +1)(\overline{X}(n+1) - \overline{X}(n))^2 X(n+1)=X(n)+n+1Xn+1​−X(n)​S2(n+1)=(1−n1​)S2(n)+(n+1)(X(n+1)−X(n))2

De esta forma el estimador de media muestral se ira calculando iteradamente hasta que el ECM sea menor que ddd.

def media_muestral(d):
    media = generar_X()
    scuad = 0
    n = 1
    while n <= 100 or sqrt(scuad/n) > d:
        n += 1
        X = generar_X()
        mediaAnt = media
        media = mediaAnt + (X - mediaAnt)/n
        scuad = (1 - 1/(n-1)) * scuad + n*(media - mediaAnt)**2

    return media

Estimador de proporción

Lo usamos cuando queremos estimar una probabilidad. Pra estimar la proporcion usamos p^=X‾(n)\hat{p} = \overline{X}(n)p^​=X(n). Teniendo en cuenta que la i-esima simulacion se tendra una variable de Bernoulli XiX_iXi​, por lo que la varianza es σ2=p(1−p)\sigma^2 = p(1-p)σ2=p(1−p) para una Bernoulli.

El n dependera de la presicion con la que querramos estimar p, esta precision es el ECM:σ2n\frac{\sigma^2}{n}nσ2​. Como dijimos que la varianza de una Bernoulli era p(1−p)p(1-p)p(1−p), entonces un estimador para la varianza de la variable simulada es:

σ^2=X‾(n)(1−X‾(n))\hat{\sigma}^2 = \overline{X}(n)(1 - \overline{X}(n)) σ^2=X(n)(1−X(n))

Luego el ECM sera : X‾(n)(1−X‾(n))n\frac{\overline{X}(n)(1 - \overline{X}(n))}{n}nX(n)(1−X(n))​

def estimador_p(d):
    p = 0
    n = 0
    while n <= 100 or sqrt(p * (1-p)/n) > d:
        n += 1
        X = generar_X()
        p = p + (X - p)/n

    return p

Estimador por intervalos

Un estimador por intervalo de un parámetro es un intervalo para el que se predice que el parámetro está contenido en él. Es decir, se tiene un intervalo aleatorio con una cierta probabilidad de contener el parámetro buscado. La confianza que se da al intervalo es la probabilidad de que el intervalo contenga al parámetro.

Estimador por intervalo de E(X) = θ\thetaθ

Queremos determinar un intervalo que contenga al parámetro θ\thetaθ con un nivel de confianza de 1−α1 - \alpha1−α. Recoramos que zαz_{\alpha}zα​ indica el numero real tal que P(Z>zα)=αP(Z > z_{\alpha}) = \alphaP(Z>zα​)=α. Ademas sabemos que:

X‾(n)−θσ/n∼N(0,1)\frac{\overline{X}(n) - \theta}{\sigma/\sqrt{n}} \sim N(0,1) σ/n​X(n)−θ​∼N(0,1)

luego como tiene una distribución simétrica podemos decir que:

P(X‾(n)−zα/2σn<θ<X‾(n)+zα/2σn)=1−αP(\overline{X}(n) - z_{\alpha/2} \frac{\sigma}{\sqrt{n}} \lt \theta \lt \overline{X}(n) + z_{\alpha/2} \frac{\sigma}{\sqrt{n}}) = 1 - \alpha P(X(n)−zα/2​n​σ​<θ<X(n)+zα/2​n​σ​)=1−α

Determina un intervalo aleatorio que contiene al parametro θ\thetaθ con una confianza de 1 - α\alphaα. Si queremos un intervalo de confianza del 95% entonces α=0.05\alpha = 0.05α=0.05 y zα/2=1.96z_{\alpha/2} = 1.96zα/2​=1.96. Y el intervalo sera: [X‾(n)−1.96σn,X‾(n)+1.96σn][\overline{X}(n) - 1.96 \frac{\sigma}{\sqrt{n}}, \overline{X}(n) + 1.96 \frac{\sigma}{\sqrt{n}}][X(n)−1.96n​σ​,X(n)+1.96n​σ​]

Cuando σ\sigmaσ es desconocido se reemplaza utilizando el estimador σ^=S2(n)\hat{\sigma} = \sqrt{S^2(n)}σ^=S2(n)​.

La longitud del intervalo esta dada por:
