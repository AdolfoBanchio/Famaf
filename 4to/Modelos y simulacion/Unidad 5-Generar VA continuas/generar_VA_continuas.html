∫−inf⁡xf(t)dt\int_{-\inf}^{x} f(t)dt ∫−infx​f(t)dt

Con esta proposicion podemos dar un algoritmo basico de como funciona el metodo de la transformada inversa:

Simulacion de variable Gamma(n, λ−1\lambda^{-1}λ−1)

Sabemos que la suma de n variables aleatorias exponenciales, independientes, con parametro λ\lambdaλ es una variable aleatoria con distribucion Gamma(n, λ−1\lambda^{-1}λ−1). Esta propiedad nos permite dar un algoritmo que a partir de exponenciales generamos una Gamma.

X=−1λlog⁡(U1⋅U2⋯Un)X = -\frac{1}{\lambda} \log(U_1 \cdot U_2 \cdots U_n) X=−λ1​log(U1​⋅U2​⋯Un​)

Y el algoritmo es:

def Gamma(n, lamda):
    # genera gamma con parametros n y 1/lamda
    u = 1
    for _ in range(n):
        u *= 1 - random()
    return -log(u)/lamda

utilizando este metodo de generacion de exponenciales para generar una gamma, entonces podemos diseñar un algoritmo para generar n variables independientes exponenciales de parametro λ\lambdaλ

def DosExp(lamda):
    V1, V2 = 1-random(), 1-random()
    t = -log(V1 * V2) / lamda
    U = random()
    X = t * U
    Y = t - X
    return X, Y

Para el caso general tenemos que calcular un unico logaritmo y n-1 uniformes adicioneles.

def Nexponenciales(n,lamda):
    t = 1

Metodo de aceptacion y rechazo

Supongamos que queremos generar X que tiene funcion de densidad f:

F(x)=P(X≤x)=∫−inf⁡xf(t)dtF(x) = P(X \leq x) = \int_{-\inf}^{x} f(t)dt F(x)=P(X≤x)=∫−infx​f(t)dt

Y que tengo un método para generar otra Y con denssidad g tq:

f(y)g(y)≤c para todo y∈R:f(y)≠0\frac{f(y)}{g(y)} \leq c \quad\text{ para todo } y \in \R : f(y) \not ={0} g(y)f(y)​≤c para todo y∈R:f(y)=0

Entonces el algoritmo de aceptacion y rechazo es:

def AceptacionRechazo():
    while True:
        y = G()
        u = random()
        if u < f(y) / (c*g(y)):
            return y

Al igual que en las variables aleatorias discretas, este metodo tiene densidad f y el numero de iteraciones es una variable geometrica con media c.

Para poder generar X rechazando contra una variable Y debemos poder acotar f(x)g(x)\frac{f(x)}{g(x)}g(x)f(x)​ con una constante c. Por lo que vamos a tener que considerar la funcion:

h(x)=f(x)g(x),para x∈R:f(x)≠0h(x) = \frac{f(x)}{g(x)}, \quad \text{para } x \in \R : f(x) \not = 0 h(x)=g(x)f(x)​,para x∈R:f(x)=0

Para esto vamos a:

  • encontrar los puntos criticos de h(x) (donde h'(x) = 0 o h(x) no esta definida)
  • ver cuales corresponden a máximos locales
  • Luego tomamos ese maximo como c

Generación de un proceso de Poisson

Recordamos que en un proceso de Poisson de parametro λ\lambdaλ los tiempos de llegada entre eventos sucesivos son exponenciales con parametro λ\lambdaλ. Entonces de esta forma si queremos generar los primeros n eventos tenemos que generar exponenciales Xi E(λ),1≤i≤nX_i ~ \Epsilon(\lambda), 1 \leq i \leq nXi​ E(λ),1≤i≤n

  • Primer evento: T1=X1T_1 = X_1T1​=X1​
  • j-esesimo evento: X1+X2+⋯+XjX_1 + X_2 + \cdots + X_jX1​+X2​+⋯+Xj​

Entonces para generar eventos hasta T, generamos evento hasta que lo generado en j+1 exceda a T. Recordamos que la forma de generar una exponencial con parametro λ\lambdaλ es −log⁡U/λ-\log{U} /\lambda−logU/λ

Entonces el codigo queda:

def eventos_poisson(lamda, T):
    t = 0
    NT = 0
    Eventos = []
    while t < T:
        U = 1 - random()
        t += - log(U) / lamda
        if t <= T:
            NT += 1
            Eventos.append(t)
    return NT, Eventos

NT es la cantidad de eventos hasta el tiempo T, y en Eventos se devuelve el tiempo cuando ocurrio el i-esimo evento.
